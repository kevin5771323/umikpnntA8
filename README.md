# 宿舍管理系统 java297

## 项目概述

宿舍管理系统是基于SpringBoot和Vue的前后端分离架构，主要实现了迁出登记、缺勤登记、宿管管理、学生管理等功能模块，确保了宿舍管理的高效和规范。

## 技术栈

### 后端

- SpringBoot
- MyBatis

### 前端

- Vue

### 数据库

- MySQL

### 技术语言与开发平台

- JDK 1.8
- 开发平台：Idea + Windows

## 运行环境

- Linux
- Windows

## 功能模块

- 迁出登记
- 缺勤登记
- 宿管管理
- 学生管理

## 系统角色

- 系统管理员
- 宿管员
- 学生

## 目录结构

```plaintext
├── backend                # 后端代码
│   ├── src
│   │   ├── main
│   │   │   ├── java
│   │   │   ├── resources
│   ├── pom.xml
├── frontend               # 前端代码
│   ├── src
│   ├── public
│   ├── package.json
└── database               # 数据库相关文件
    ├── schema.sql         # 数据库结构
    └── data.sql           # 数据库数据
```

## 部署步骤

1. 环境准备：确保Java 1.8和MySQL环境已安装。
2. 数据库部署：导入`database/schema.sql`以建立数据库结构，导入`database/data.sql`以初始化数据。
3. 后端部署：进入`backend`目录，执行`mvn install`进行编译打包，然后运行jar包。
4. 前端部署：进入`frontend`目录，执行`npm install`安装依赖，然后执行`npm run build`构建前端资源，最后将构建结果部署到Web服务器。
